## Description

Solution for SignalWire's coding challenge.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# 1. Initialize DB 
start:dev:db

# 2. Start Server
$ npm run start

# 3. Start Server in hot-reload mode
$ npm run start:dev

```

## Test

```bash
# e2e tests
$ npm run test
```
